# Output Model

The public collector writes a bundle with four release-oriented artifacts:

## `rent_benchmarks.csv`

Collector-produced benchmark observations suitable for downstream publication.

## `locality_crosswalk.csv`

Locality metadata used by the public release process.

## `source_inventory.csv`

Machine-readable source inventory generated from the public source registry.

## `manifest.json`

Public-safe metadata describing the bundle contents, checksums, and source summary.
